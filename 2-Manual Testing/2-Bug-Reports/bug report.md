# BUG-001 – Sample: Incorrect error message on invalid login

## Environment
- Site: https://www.saucedemo.com/
- Browser: Chrome 120
- OS: Windows 11

## Steps to Reproduce
1. Open the login page
2. Enter username: standard_user
3. Enter password: wrong_password
4. Click Login

## Expected Result
A clear error message indicating invalid credentials.

## Actual Result
Error message is unclear / does not specify credentials issue. (Sample)

## Severity / Priority
- Severity: Medium (user can’t understand what went wrong)
- Priority: Medium
